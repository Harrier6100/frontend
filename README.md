- コンポーネント
    - コンポーネントはvite.config.jsでインポート
    - ページおよびコンポーネントで個別にインポートはしない
- 翻訳
    - 翻訳ファイルはi18n/localesに保存
    - デフォルトの言語は"ja"
    - キーが未登録の場合、キーを表示
    - 対応言語を増やす場合、localesにxx.jsonを追加
    - 翻訳がない場合、フォールバックせず、missingだけで対応
    - フォールバックの警告は非表示
    - レガシーモードは無効
- ルーティング
    - ルーティングはrouter/routesに定義、router/index.jsでインポート
    - 基本的に、viewsのディレクトリ構成とリンク
- メニュー
    - メニューはconfig/menuに定義、components/nav/Sidebar.vueでインポート
